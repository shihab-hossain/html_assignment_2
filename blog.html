In JavaScript, var, let, and const are used to declare variables, but they have different scopes and behaviors.

var......
variables declared with var are function-scoped.it means when we use var in JavaScript it become a global variable and it is accessible the entire program.

let......
Variables declared with let have block scope, it mean they are only accessible within the block where they are defined.

const....
Variables declared with const also have block scope.const is used for constant and when a value is assigned to a const variable it cannot be changed.

In programming null and undefined are both special:

null...
when we assign the value of a variable is null it means the variable has no meaningful value.sometime a function can not return meaningful value in this case function return null.

undefined....
when a variable is declared but not assigned a value, it is automatically assigned the value undefined.it means the variable has no value.
